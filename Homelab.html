My Home Lab Infrastructure

A multi-node virtualization cluster with segmented networking, ZFS storage, and self-hosted services. This page breaks down nodes, storage pools, VLANs, automations, and backups.

Explore the Stack

Infrastructure

Five physical nodes, each with a focused role.

Node 1 — Storage & Media

Specs: AMD Ryzen 5 3600, 80 GB RAM, RTX 3060.

Roles: TrueNAS SCALE, Plex (GPU), utility VMs (Pi-hole, HA integrations, Tautulli).

Node 2 — Networking & Firewall

Specs: Intel i9-9900K, 16 GB RAM, dual 10GbE.

Roles: OpenWRT (PPPoE) → OPNsense (VLANs, DHCP, firewall).

Node 3 — Monitoring & Backups

Specs: Intel Alder Lake N97, 12 GB RAM, dual LAN.

Roles: Tailscale, Monit, Nextcloud, USB backup targets.

Node 4 — Cameras & Web

Specs: AMD Ryzen 5 3600, 32 GB RAM, RTX 3060, 10GbE.

Roles: Frigate NVR, Obico, Nginx (this site).

Node 5 — Offsite

Specs: Dell OptiPlex 390.

Roles: Offsite backups, HA, Jellyfin, monitoring (Wyse 3040 link).

Storage

TrueNAS SCALE manages multiple ZFS pools. SAS lives on Node 1; selected SATA drives are passed through from Node 4.

PoolConfigurationUsableLocationPurpose
Pool A 6 × 6TB SAS (RAIDZ2) ~24 TB Node 1 School (250GB+), critical Plex (~2TB), room to grow w/ redundancy
Pool B 1 × 12TB SAS (single) 12 TB Node 1 Bulk Plex (cartoons/movies) — re-downloadable
Pool C 2 × 2TB SATA (striped) ~4 TB Node 4 → TrueNAS Scratch / torrent staging (no redundancy)
Pool D 2 × 4TB SATA (mirror) ~4 TB Node 4 → TrueNAS Photo archive (~3.4 TB) — redundant
